Content Intelligence provides structured CLI access to Drupal entity data and aggregated intelligence from multiple sources. Designed for AI tools, coding assistants, and automation systems.

Built for AI Integration

AI tools and automation systems need structured access to content data. Content Intelligence provides a CLI interface for:

  • AI Coding Assistants — Tools like Claude Code, GitHub Copilot, and Cursor can query site structure, content types, and field schemas
  • Chatbots — Access content data to provide contextual responses about your website
  • Content Auditing — Retrieve entity data with analytics from multiple plugins
  • Automation — Process content programmatically via Drush commands

Architecture

  • Entity-agnostic — Works with all content entity types (nodes, taxonomy terms, media, users, paragraphs, etc.)
  • JSON output — Machine-readable format for tool integration
  • CLI-native — Drush commands for scripting and AI tool execution
  • Plugin system — Extensible via PHP 8 Attributes

Drush Commands

All commands support --format=json for machine-readable output.

Command Description
drush ci:types List available entity types
drush ci:bundles [type] List bundles for an entity type
drush ci:fields [type] [bundle] List fields for an entity type/bundle
drush ci:plugins List available intel plugins
drush ci:list [type] [bundle] List entities of a given type
drush ci:entity [type] [id] Get full intelligence data for an entity
drush ci:batch [type] Batch collect intel for multiple entities

Intelligence Plugins

Built-in:

  • Statistics — Page view counts from the Statistics module
  • Content Translation — Translation status and coverage metrics

Contrib Integrations:

  • Analyze — Content analysis data (readability, SEO, etc.)
  • AI Social Posts — Social media posts linked to content

Custom Plugins

Add custom intelligence sources using PHP 8 Attributes:

<?php

#[ContentIntel(
  id: 'my_plugin',
  label: new TranslatableMarkup('My Plugin'),
  description: new TranslatableMarkup('Custom intelligence data.'),
  entity_types: ['node'],
  weight: 50,
)]
class MyPlugin extends ContentIntelPluginBase {

  public function collect(ContentEntityInterface $entity): array {
    return [
      'score' => 0.95,
      'status' => 'published',
    ];
  }

}

Requirements

  • Drupal 10.3+ or 11.x
  • PHP 8.1+
  • Drush 12+

Optional Dependencies

  • Statistics — view count tracking
  • Content Translation — translation status
  • Analyze — content analysis plugins
  • AI Social Posts — social post tracking

Quick Start

  1. Install: composer require drupal/content_intel
  2. Enable: drush en content_intel
  3. Configure plugins at /admin/config/content/content-intel
  4. Test: drush ci:entity node 1 --format=json
